# Lucknow Census 2011 – Demographic Analysis Dashboard (Excel Project)

This project presents an interactive Excel dashboard constructed using data from the 2011 Indian Census for the city of Lucknow. The dashboard offers a visual and analytical overview of key demographic indicators, enabling a deeper understanding of the city’s population composition and social characteristics.

# Key Features

Population Analysis: Breakdown of urban and rural population distribution within Lucknow.

Education Insights: Visualization of literacy rates and educational attainment across different groups.

Marital Status Patterns: Proportion of married, unmarried, and divorced individuals represented visually.

Child Survival Statistics: Analysis of surviving children across various demographics.

Interactive Dashboard: Consolidated dashboard summarizing all indicators using charts, slicers, and KPIs.

# Tech Stack

Tool: Microsoft Excel

Techniques: Pivot Tables, Charts, Slicers, Conditional Formatting, and Data Visualization

# Objective

To analyze and visualize key demographic aspects of Lucknow using Census 2011 data, enabling easy interpretation of social and population trends through a clean, interactive Excel dashboard.
